## 状态值定义

### YAML 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `requirement_name` | string | 需求名称 |
| `current_phase` | string | 当前阶段（P1-P16） |
| `status` | enum | 当前状态 |
| `created_at` | date | 创建日期 |
| `updated_at` | date | 最后更新日期 |

### 状态枚举

| 状态值 | 说明 | 图标 |
|--------|------|------|
| `pending` | 待执行 | ⏳ |
| `in_progress` | 进行中 | 🔄 |
| `completed` | 已完成 | ✅ |
| `blocked` | 已阻塞 | ❌ |
| `revision` | 需要修改 | 🔙 |
