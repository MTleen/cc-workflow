# cc-workflow 项目状态

## 项目概述

**项目名称**：Claude Code 中心化团队工作流系统

**目标**：以 Claude Code 为中心重构团队工作流，实现从产品设计、前端设计到开发、测试、上线的全流程自动化。

**核心原则**：团队成员只需**提需求、做评审、反馈意见**，其他工作由 Claude Code 自动完成。
